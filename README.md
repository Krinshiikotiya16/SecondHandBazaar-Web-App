# 🛒  Second-Hand Marketplace Web Application

## 📌 About the Project
SecondHand-Bazaar is a web-based marketplace application that allows users to buy and sell pre-owned products online. The platform provides a simple and secure environment where users can create accounts, list products, search available items, make purchases, and complete transactions through an integrated payment gateway.

The project was developed to demonstrate practical implementation of web development concepts including authentication systems, database management, form validation, email integration, payment gateway integration, and REST API concepts.

## 🎯 Project Objective

The objective of this project is to provide a user-friendly platform for buying and selling second-hand products while demonstrating practical web development concepts such as authentication, database connectivity, payment integration, email services, security validation, and REST API implementation.

## 📜 Conclusion
